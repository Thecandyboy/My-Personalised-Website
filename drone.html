About the Project

This is a course project for the Makerspace course, supervised by Professor Kushal Tuckley

In a team of four, we designed, assembled, and tested a Wi-Fi-controlled drone using a mobile device

We employed Fusion360 for prototyping, along with 3D printing and laser-cutting techniques

To ensure safe and stable flight, we developed a PID control algorithm and utilized the RemoteXY mobile app for enhanced user control and responsiveness
